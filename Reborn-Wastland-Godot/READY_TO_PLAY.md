## ğŸ› Troubleshooting

### Models not showing
- Check asset path: `res://assets/Survival 3D/`
- Verify FBX files are present
- Check console for loading errors

### Performance issues
- Lower quality preset manually
- Reduce max_zombies count
- Disable shadows: set shadow_enabled = false

### Sound not working
- Sound files in `assets/Survival 3D/Sounds/`
- Check AudioManager bus connections
- Verify AudioStreamPlayer nodes exist

### NPC/Zombie not spawning
- Check WorldBuilder console output
- Verify scene instantiation in GameManager_v2
- Check collision shapes are created
